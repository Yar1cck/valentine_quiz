# Love Quiz — романтическая викторина для пар

Небольшой сайт на **Python (Flask) + JavaScript** с романтическим дизайном ко Дню святого Валентина.

## Возможности

- **Конструктор викторин** — создавайте свои викторины с любыми вопросами и вариантами ответов.
- **Сравнение ответов партнёров** — сначала один партнёр проходит тест, затем второй. Система сравнивает ответы и показывает процент совпадений.
- Готовые викторины:
  - «Насколько хорошо ты знаешь партнёра» — проверка знания привычек и предпочтений партнёра.
  - «Тест на совместимость» — сравнение взглядов и привычек пары.
  - «Кто вы из фильма» — романтический тест с кадрами из фильмов.
- Страница **Галерея**, которая показывает фотографии из папки `images` в корне проекта.
- Адаптивный дизайн: удобно как на десктопах, так и на телефонах.

## Запуск проекта

1. Убедитесь, что у вас установлен Python 3.10+.
2. В корне проекта (папка `Викторина`) установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

3. Запустите приложение:

   ```bash
   python app.py
   ```

4. Откройте браузер и перейдите по адресу:

   ```text
   http://127.0.0.1:5000
   ```

## Галерея

1. В корне проекта создайте папку `images`, если её ещё нет.
2. Скопируйте в неё ваши общие фотографии (`.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`).
3. Откройте страницу «Галерея» в меню сайта — изображения появятся автоматически.

## Картинки результатов и личные изображения

### 1. Личные фотографии для галереи

- Папка: `images/` (в корне проекта).
- Сюда кладите любые ваши общие фотографии — они появятся на вкладке «Галерея».

### 2. Личные картинки для результатов викторин

Созданы папки:

- `static/images/results` — картинки для результатов текстовых тестов.
- `static/images/movies` — кадры из фильмов для кино‑викторины.
- `personal_images` — дополнительная папка, куда вы можете складывать любые личные изображения (например, свои варианты картинок для результатов), не привязанные к галерее.

Чтобы картинки появлялись в результатах:

1. Для теста «Насколько хорошо ты знаешь партнёра» положите в `static/images/results` файлы:
   - `know_expert.jpg`
   - `know_good.jpg`
   - `know_start.jpg`

2. Для теста «Тест на совместимость» положите в `static/images/results` файлы:
   - `compat_high.jpg`
   - `compat_middle.jpg`
   - `compat_low.jpg`

3. Для кино‑викторины положите в `static/images/movies` кадры из фильмов с такими именами:
   - `fault_in_our_stars.jpg` — «Виноваты звёзды»
   - `me_before_you.jpg` — «До встречи с тобой»
   - `bridget_jones.jpg` — «Дневник Бриджит Джонс»
   - `notebook.jpg` — «Дневник памяти»
   - `la_la_land.jpg` — «Ла-Ла Ленд»

После добавления изображений просто перезагрузите страницу с викториной — картинки подтянутся автоматически.

## Структура проекта

- `app.py` — основной файл Flask‑приложения.
- `templates/` — HTML‑шаблоны (`base.html`, `index.html`, `quiz.html`, `gallery.html`).
- `static/css/styles.css` — стили и анимации.
- `static/js/main.js` — логика викторины на JavaScript.
- `images/` — ваши фотографии для галереи (папку нужно создать вручную).
